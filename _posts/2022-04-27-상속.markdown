# 상속
클래스를 정의할때 이미 구현된 클래스를 상속받아서   
속성, 메서드를 쓸수있는 클래스를 구현(@Override)   
<br/>

# 상속 문법
class B extends A {}   
A로부터 B가 상속받음   
extends뒤에는 단 하나의 class만 사용가능   
B가 A보다 더 구체적인 의미
